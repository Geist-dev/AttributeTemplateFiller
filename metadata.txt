about=Attribute Templates & Autofill allows you to create reusable attribute presets for vector layers and automatically apply them when creating new features.
 The plugin significantly speeds up digitizing workflows and ensures consistent attribute data entry.
 
 Main features:
 • Create multiple attribute templates per layer
 • Automatically apply active template when adding new features
 • Safe handling of primary key fields (prevents UNIQUE constraint errors)
 • Apply templates to selected features
 • Import and export templates (JSON format)
 • Multilingual interface
 
 This plugin is ideal for cadastral mapping, utilities, environmental surveys, infrastructure projects, and large-scale GIS data entry tasks.
 
 Compatible with QGIS 3.16+
